Implementation of the aggregated system

We consider an age structured population with two age classes: juvenile (immature) and adult (mature) individuals. The structured population is further assumed to be composed of two local populations coupled by dispersal. We denote \(J_{i}\) and \(A_{i}\), respectively, the densities of juveniles and adults in patch \(i=1,2\). The population vector is thus: \[\vec{N}=\left(J_{1},J_{2},A_{1},A_{2}\right)^{T},\] where \(T\) denotes transposition. We also use the following notation \[ J=J_{1}+J_{2},\qquad A=A_{1}+A_{2}, \qquad \vec{n}=(J,A)^T \] for the total number of juveniles and adults individuals.

The aggregated system (7) in the paper is \[ \vec{n}(t+1)\, = \,\mathcal L(\vec{n}(t))\,\vec{n}(t) \] Here \(\vec n(t)\) is the vector giving the total number of juvenile and adults in the population at time \(t\).

The matrix \(L(\vec{n}(t))\) of the aggregated system is given by:

\[ {\cal L}(\vec{n}(t)) \, = \, \left( \begin{array}{cc} 0 & \bar \phi(\vec n(t)) \\ \bar \sigma_J & \bar \sigma_A \\ \end{array} \right)\vec{n}(t) \] where: \[ \begin{cases} \bar\phi(\vec n(t)) = \bar\phi(J,A)=\mu_A\phi_1(J,A)+ (1-\mu_A)\phi_2(J,A),\\[3mm] \bar \sigma_J=\mu_J\sigma_J^1+ (1-\mu_J)\sigma_J^2\\[3mm] \bar \sigma_A= \mu_A\sigma_J^1+ (1-\mu_A)\sigma_J^2. \end{cases} \]

Let us describe in detail the components entering this equations, and the way they are implemented here:

Dispersal Rates

Juvenile \(p_i^J\) and adult \(p_i^A\) dispersal rates (that is, the fraction of individuals of each age class leaving patch \(i\)) are supposed to be constant for \(\, i=1 , 2\). For this example we take the following particular values:

p1J = .4 
p2J = .4
p1A = .4
p2A = .4

From these values we obtain \(\mu_J\) and \(\mu_A\), the asymptotic distribution of juvenile and adult individuals at patch 1 due exclusively to dispersal:

\[\mu_J =\dfrac{p_2^J}{p_1^J + p_2^J}\] \[\mu_A =\dfrac{p_2^A}{p_1^A + p_2^A}\]

This is implemented as follows:

(muJ = p2J / (p1J + p2J))
## [1] 0.5
(muA = p2A / (p1A + p2A))
## [1] 0.5

and both values are collected in a vector called mu.

(mu = c(muJ, muA))
## [1] 0.5 0.5

Survival Rates

Juvenile and adult survival rates are given rexpectively by \(\,\sigma_i^J, \sigma_i^A\) for \(i=1, 2\). They are also considered to be constant in the model, and we take the following particular values:

sigmaJ1 = .2 
sigmaJ2 = .75 
sigmaA1 = .85 
sigmaA2 = .3

Then the values of \(\bar\sigma_J\) and \(\bar\sigma_A\) are given by:

barSigmaJ = function(mu)  sigmaJ1 * mu[1] + sigmaJ2 * (1 - mu[1])
   
barSigmaA = function(mu)  mu[2] * sigmaA1 + (1 - mu[2]) * sigmaA2

Fertility rates

In this model, the fertility rates in both patches are given by functions \(\,\phi_i\in C^1\left(\bar{\mathbb R}_+^2,\mathbb R_+\right)\) for \(i=1, 2\). We consider three possible classes of fertility functions, as discussed in the paper.

For a concrete example of this construction, we will consider the following class C1 functions, for any choice of \(\,a_1, a_2>0\): \[ \begin{cases} \phi_1(J, A)=\psi_1\exp(-a_1(\mu_J J + \mu_A A) ) \\[3mm] \phi_2(J, A)=\psi_2\exp(-a_2((1 -\mu_J) J + (1 - \mu_A) A) ) \end{cases} \] For the case of class C2 fertility functions we will use this concrete example: \[ \begin{cases} \phi_1(J, A)=\psi_1\cdot (1 + \mu_J \mu_A J A) \exp(-a_1(\mu_J J + \mu_A A) ) \\[3mm] \phi_2(J, A)=\psi_2\cdot (1 + (1 - \mu_J) (1 - \mu_A) J A) \exp(-a_2((1 -\mu_J) J + (1 - \mu_A) A) ) \end{cases} \] And, finally, for the case of class C3 fertility functions we will use this concrete example: \[ \begin{cases} \phi_1(J, A)=\psi_1\cdot (1 + \mu_A A) \exp(-a_1(\mu_J J + \mu_A A) ) \\[3mm] \phi_2(J, A)=\psi_2\cdot (1 + (1 - \mu_A) A) \exp(-a_2((1 -\mu_J) J + (1 - \mu_A) A) ) \end{cases} \] The following code implements the above examples of fertility functions. The variables phi1Class and phi2Class can be used to select the fertility function class for each of the examples that we will examine:

phi1 = function(J, A, psi1, mu, a){
  if(phi1Class == 1){
    psi1 * exp(-a[1] * (J * mu[1] + A * mu[2])) 
  } else if(phi1Class == 2){
    psi1 *(1 + J * mu[1] * A * mu[2]) * exp(- a[1] * (J * mu[1] + A * mu[2]))
  } else if(phi1Class == 3){
    psi1 * (1 + A * mu[2]) * exp(-a[1] * (J * mu[1] + A * mu[2]))   
  }
}


phi2 = function(J, A, psi2, mu, a){
  if(phi2Class == 1){
    psi2 * exp(-a[2] * (J * (1 - mu[1]) +  A * (1 - mu[2]))) 
  } else if(phi2Class == 2){
    psi2 * (1 + J * (1 - mu[1]) * A * (1 - mu[2])) * exp(-a[2] * (J * (1 - mu[1]) + A * (1 - mu[2]))) 
  } else if(phi2Class == 3){
    psi2 * (1 + A * (1 - mu[2])) * exp(-a[2] * (J * (1 - mu[1]) + A * (1 - mu[2])))   
  }
}

The aggregated system matrix

With the above ingredients we are ready to implement the elements of the aggregated system matrix \(\ \mathcal L(\vec{n}(t))\). We begin with \[\bar\phi(J,A) = \mu_A \phi_1(J,A) + (1-\mu_A) \phi_2(J,A)\]

This is implemented as:

barPhi = function(J, A, psi, mu, a){
  mu[2] * phi1(J, A, psi[1], mu, a) + 
  (1 - mu[2]) * phi2(J, A,  psi[2], mu, a)    
}

The action of the aggregated system matrix on the vector \(\,\vec n(t) = (J(t), A(t))^T\) is then described by the following code:

RedSys = function(J, A, psi, mu,a){
  return(c(A * barPhi(J, A, psi, mu, a),  
           J * barSigmaJ(mu) + A * barSigmaA(mu)))
}

Iterations loop and bifurcation diagram for the aggregated system

The goal of this code is to obtain a bifurcation diagram for this reduced system. In order to do this we will obtain a large number of iterations of the system. The initial iterations are then discarded and we plot the diagram using only the last ones. Thus, the total number of iterations is obtained as the sum as the number of initial, or transient, iterations plus the number of iterations that will appear in the plot.

The default numbers used in this example are \(\,500\) transient iterations and \(\,200\) plotted iterations, but these can be easily changed in the code below.

To obtain the diagram we will plot the total size of the population, given by the scalar \[n(t) = J(t) + A(t)\] against the values of the bifurcation parameter. In the following analysis we will consider two different bifurcation parameters, representing different choices of magnitudes that can be used to describe or manage the population dynamics:

  • The net reproductive number \(\nu\) PUEDE HABER CAMBIADO (see Equation (9), page 8 in the paper). Here we will consider a set of values of the net reproductive number parametrized by the coefficient \(\psi_1\) of the fertility function \(\phi_1\) as discussed before.

  • The asymptotic distribution of juvenile individuals at patch 1, that is, \(\mu_J\).

Now, for each value of the bifurcation parameter we will get a set of iterates. Each column of the matrices \(M_J\) and \(M_a\) will contain, respectively, the iterates of \(J\) and \(A\) for a given value of the bifurcation parameter.

We initialize the population values by setting the value of the first column of both \(M_J\) and \(M_A\). Again, we have selected \(5\) juvenile and \(5\) adult individuals as initial values, but these values can be easily changed in the code.

The iteration is carried out by two nested for loops. The iteration variables r and c indicate, respectively, the row and column of the matrices \(M_J\) and \(M_A\) being considered in each iteration. The result of each iteration of the inner loop is the vector \(\vec n(t+1) = (J(t+1), A(t+1))\). After the inner loop is concluded, in the bifurcation parameter is the net net reproductive number, we use the values of \(\,\psi_1\) and \(\, \psi_2\) to obtain the value of \(\nu\) for that iteration of the outer loop.

After the iteration loops conclude, we get the total population number for each value of the bifurcation parameter simply by adding the matrices \(M_J\) and \(M_A\): \[M_N = M_A + M_J\] We will, however, only use the final set of iterates. Therefore, we discard the initial or transient iterations when adding the matrices.

Now, after the iteration loop concludes, we can simply plot the columns of matrix \(\,M_N\) against the values of the bifurcation parameter \(\nu\). The matplot function in R does precisely that.

These operations are implemented in the iterLoop function below.

iterLoop = function(transIter, plotIter, paramIter, 
                    J0=5, A0=5, psi, mu, a){
  
  K = length(paramIter)
  iterPerValue = transIter + plotIter
  iterRange = 2:iterPerValue
  
  mJ = matrix(rep(0, as.integer(K * iterPerValue)), 
              ncol= iterPerValue)
  mA = matrix(rep(0, as.integer(K * iterPerValue)), 
              ncol= iterPerValue)
  
  mA[ , 1] = J0
  mJ[ , 1] = A0
  
  nu = numeric(K)
  
  for(r in 1:K){
    for(c in iterRange){  
      nextIter = RedSys(mJ[r, c-1], mA[r, c-1], psi[r, ], mu[r, ],a)  
      mJ[r, c] = nextIter[1]
      mA[r, c] = nextIter[2]
      }  
    nu[r] = barSigmaJ(mu[r, ]) * barPhi(0, 0, psi[r, ], mu[r, ], a) / 
    (1 - barSigmaA(mu[r, ]))
    }
  
  mN = mJ[, -(1:transIter)] + mA[, -(1:transIter)]
  
  results = list(nu, mN)
  return(results)
  
}

Persistence and dispersal

The global net reproductive number depends on several parameter of the model and, in particular, on the asymptotic distribution of juvenile and adult individuals, \(\mu_J\) and \(\mu_A\) respectively. In the following figures we plot the global net reproductive as \(\mu_J\) and \(\mu_A\) vary for different patches configurations. We first define a specific function to calculate \(\nu\) for each values of \(\mu_J\) and \(\mu_A\). nuFunction=function(paramIter, psi, mu, a){

  nu = numeric(K)
  for(r in 1:K){ 
    nu[r] = barSigmaJ(mu[r, ]) * barPhi(0, 0, psi, mu[r, ], a) / 
    (1 - barSigmaA(mu[r, ]))
    }
    results = nu
  return(results)
}

Dispersal and bifurcation diagram when \(\nu\) is close to doubling period threshold value

It can be calculated by simply calling the above defined funcion iterLoop. Note that \(\psi_1\) and \(\psi_2\) are now fixed values and the bifurcation parameter is \(\nu_J\). In the left panel we assume

phi1Class = 1
phi2Class = 1

sigmaJ1 = .2 
sigmaJ2 = .6 
sigmaA1 = .85 
sigmaA2 = .3

mu=seq(0, 1, by = 0.01)
psi1 = rep(x=30, times = length(mu))
alpha = .5
psi2 = alpha * psi1

psiMatrix = cbind(rep(psi1, length(mu)), rep(psi2, length(mu)))
psi = cbind(psi1, psi2)

par(mfrow = c(1, 2))

muMatrix=cbind(mu, mu)
iterations = iterLoop(transIter = 500, plotIter = 200, paramIter = mu,  
                      J0 = 5, A0 = 5, psi, muMatrix, a = c(.17, .2))
matplot(mu, iterations[[2]], type="l", col="black", xlab="nu", ylab="Total population")

  
muMatrix=cbind(mu, 1-mu)
iterations = iterLoop(transIter = 500, plotIter = 200, paramIter = mu,  
                      J0 = 5, A0 = 5, psi, muMatrix, a = c(.17, .2))
matplot(mu, iterations[[2]], type="l", col="black", xlab="nu", ylab="Total population")
